112: <h1>PHP Fine Diff</h1>
113: <div>
114:     <p style="margin-right:5em">This page demonstrate the <a href="finediff-code.php"><code>FineDiff</code></a> class
115:         (as in &ldquo;<b>fine</b> granularity <b>diff</b>&rdquo;) I wrote &ndash; starting from scratch &ndash; to
116:         generate a <u>lossless</u> (won't eat your line breaks), <u>compact</u> opcodes string listing the sequence of
117:         atomic actions (copy/delete/insert) necessary to transform one string into another (thereafter referred as
118:         the &ldquo;From&rdquo; and &ldquo;To&rdquo; string). The &ldquo;To&rdquo; string can be rebuilt by running the
119:         opcodes string on the &ldquo;From&rdquo; string. The <code>FineDiff</code> class allows to specify the
120:         granularity, and up to character-level granularity is possible, in order to generate the smallest diff possible
121:         (at the <i>potential</i> cost of increased CPU cycles.)</p>
122:     <p>Typical usage:</p>
123:     <p class="code">
124:         include '<a href="finediff-code.php">finediff.php</a>';<br>
125:         $opcodes = FineDiff::getDiffOpcodes($from_text, $to_text /* , default granularity is set to character */);<br>
126:         // store opcodes for later use...</p>
127:     <p>Later, <code>$to_text</code> can be re-created from <code>$from_text</code> using <code>$opcodes</code> as
128:         follow:</p>
129:     <p class="code">
130:         include '<a href="finediff-code.php">finediff.php</a>';<br>
131:         $to_text = FineDiff::renderToTextFromOpcodes($from_text, $opcodes);
132:     </p>
133:     <p>Try it by inserting your own text, or <a href="viewdiff-ex?sample=1">Use sample text</a>, or <a
134:             href="viewdiff-ex">Start from scratch</a>, or just use the plain <a href="viewdiff.php">Online diff
135:             viewer</a>:</p>
136:     <?php
137:     // http://www.php.net/manual/en/function.get-magic-quotes-gpc.php#82524
138:     function stripslashes_deep(&$value)
139:     {
140: 
141:         $value = is_array($value) ? array_map('stripslashes_deep', $value) : stripslashes($value);
142:         return $value;
143:     }
144: 
145:     if (( function_exists("get_magic_quotes_gpc") && get_magic_quotes_gpc() ) || ( ini_get('magic_quotes_sybase') && strtolower(ini_get('magic_quotes_sybase')) != "off" )) {
146:         stripslashes_deep($_GET);
147:         stripslashes_deep($_POST);
148:         stripslashes_deep($_REQUEST);
149:     }
150: 
151:     require_once 'finediff.php';
152: 
153:     $from = '';
154:     $to = '';
155:     $granularity = 2;
156:     if (isset( $_REQUEST['granularity'] ) && ctype_digit($_REQUEST['granularity'])) {
157:         $granularity = max(min(intval($_REQUEST['granularity']), 3), 0);
158:     }
159:     $rendered_diff = '';
160:     if (!empty( $_REQUEST['from'] ) || !empty( $_REQUEST['to'] )) {
161:         if (!empty( $_REQUEST['from'] )) {
162:             $from = $_REQUEST['from'];
163:         }
164:         if (!empty( $_REQUEST['to'] )) {
165:             $to = $_REQUEST['to'];
166:         }
167:     } else {
168:         if (!empty( $_REQUEST['sample'] )) { // use sample text?
169:             // Sample text:
170:             // http://en.wikipedia.org/w/index.php?title=Universe&action=historysubmit&diff=414830579&oldid=378547111
171:             $from = file_get_contents('sample_from.txt');
172:             $to = file_get_contents('sample_to.txt');
173:         }
174:     }
175:     $from_len = strlen($from);
176:     $to_len = strlen($to);
177:     $use_stdlib_diff = !empty( $_REQUEST['stdlib'] ) && ctype_digit($_REQUEST['stdlib']) && intval($_REQUEST['stdlib']) === 1;
178: 
179:     require_once 'Text/Diff.php';
180: 
181:     $start_time = gettimeofday(true);
182:     if ($use_stdlib_diff) {
183:         if ($granularity < 3) {
184:             $delimiters = array(
185:                 FineDiff::paragraphDelimiters,
186:                 FineDiff::sentenceDelimiters,
187:                 FineDiff::wordDelimiters,
188:                 FineDiff::characterDelimiters
189:             );
190:             function extractFragments($text, $delimiter)
191:             {
192: 
193:                 $text = str_replace(array("\n", "\r"), array("\1", "\2"), $text);
194:                 $delimiter = str_replace(array("\n", "\r"), array("\1", "\2"), $delimiter);
195:                 if (empty( $delimiter )) {
196:                     return str_split($text, 1);
197:                 }
198:                 $fragments = array();
199:                 $start = $end = 0;
200:                 for (; ;) {
201:                     $end += strcspn($text, $delimiter, $end);
202:                     $end += strspn($text, $delimiter, $end);
203:                     if ($end === $start) {
204:                         break;
205:                     }
206:                     $fragments[] = substr($text, $start, $end - $start);
207:                     $start = $end;
208:                 }
209:                 return $fragments;
210:             }
211: 
212:             $from_fragments = extractFragments($from, $delimiters[$granularity]);
213:             $to_fragments = extractFragments($to, $delimiters[$granularity]);
214:             $diff = new Text_Diff('native', array($from_fragments, $to_fragments));
215:             $exec_time = sprintf('%.3f sec', gettimeofday(true) - $start_time);
216:             $edits = array();
217:             ob_start();
218:             foreach ($diff->getDiff() as $edit) {
219:                 if ($edit instanceof Text_Diff_Op_copy) {
220:                     $orig = str_replace(array("\1", "\2"), array("\n", "\r"), implode('', $edit->orig));
221:                     $edits[] = new fineDiffCopyOp(strlen($orig));
222:                     echo htmlentities($orig);
223:                 } else {
224:                     if ($edit instanceof Text_Diff_Op_delete) {
225:                         $orig = str_replace(array("\1", "\2"), array("\n", "\r"), implode('', $edit->orig));
226:                         $edits[] = new fineDiffDeleteOp(strlen($orig));
227:                         echo '<del>', htmlentities($orig), '</del>';
228:                     } else {
229:                         if ($edit instanceof Text_Diff_Op_add) {
230:                             $final = str_replace(array("\1", "\2"), array("\n", "\r"), implode('', $edit->final));
231:                             $edits[] = new fineDiffInsertOp($final);
232:                             echo '<ins>', htmlentities($final), '</ins>';
233:                         } else {
234:                             if ($edit instanceof Text_Diff_Op_change) {
235:                                 $orig = str_replace(array("\1", "\2"), array("\n", "\r"), implode('', $edit->orig));
236:                                 $final = str_replace(array("\1", "\2"), array("\n", "\r"), implode('', $edit->final));
237:                                 $edits[] = new fineDiffReplaceOp(strlen($orig), $final);
238:                                 echo '<del>', htmlentities($orig), '</del>';
239:                                 echo '<ins>', htmlentities($final), '</ins>';
240:                             }
241:                         }
242:                     }
243:                 }
244:             }
245:             $rendered_diff = ob_get_clean();
246:             $rendering_time = sprintf('%.3f sec', gettimeofday(true) - $start_time);
247:         } // character-level granularity not allowed
248:         else {
249:             $edits = false;
250:             $exec_time = '?';
251:             $rendering_time = '?';
252:             $rendered_diff = '<span style="color:gray">Character-level granularity not allowed when using <code>Text_Diff</code>, due to performance issues.</span>';
253:         }
254:     } else {
255:         $granularityStacks = array(
256:             FineDiff::$paragraphGranularity,
257:             FineDiff::$sentenceGranularity,
258:             FineDiff::$wordGranularity,
259:             FineDiff::$characterGranularity
260:         );
261: 
262:         $diff = new FineDiff($from, $to, $granularityStacks[$granularity]);
263:         $edits = $diff->getOps();
264:         $exec_time = sprintf('%.3f sec', gettimeofday(true) - $start_time);
265:         $rendered_diff = $diff->renderDiffToHTML();
266:         $rendering_time = sprintf('%.3f sec', gettimeofday(true) - $start_time);
267:     }
268: 
269:     if ($edits !== false) {
270:         $opcodes = array();
271:         $opcodes_len = 0;
272:         foreach ($edits as $edit) {
273:             $opcode = $edit->getOpcode();
274:             $opcodes_len += strlen($opcode);
275:             $opcode = htmlentities($opcode);
276:             if ($edit instanceof FineDiffCopyOp) {
277:                 $opcodes[] = "{$opcode}";
278:             } else {
279:                 if ($edit instanceof FineDiffDeleteOp) {
280:                     $opcodes[] = "<span class=\"del\">{$opcode}</span>";
281:                 } else {
282:                     if ($edit instanceof FineDiffInsertOp) {
283:                         $opcodes[] = "<span class=\"ins\">{$opcode}</span>";
284:                     } else /* if ( $edit instanceof FineDiffReplaceOp ) */ {
285:                         $opcodes[] = "<span class=\"rep\">{$opcode}</span>";
286:                     }
287:                 }
288:             }
289:         }
290:         $opcodes = implode("", $opcodes);
291:         $opcodes_len = sprintf('%d bytes (%.1f %% of &quot;To&quot;)', $opcodes_len,
292:             $to_len ? $opcodes_len * 100 / $to_len : 0);
293:     } else {
294:         $opcodes = '?';
295:         $opcodes_len = '?';
296:     }

370: <div id="notes" style="font:11px sans-serif"><h3 style="margin-top:0">Notes</h3>
371:     <p>The PHP-based engine of <code><a href="http://pear.php.net/package/Text_Diff/">Text_Diff</a></code> is forced, in
372:         order to meaningfully compare results with PHP-based <code>FineDiff</code>. <code>Text_Diff</code> is naturally
373:         geared toward line-level granularity, and to compute diff for a higher granularity (sequences, words,
374:         characters), line break characters (\n, \r) are replaced in order to avoid having <code>Text_Diff</code> from
375:         eating our line breaks &mdash; so extra steps are required.</p>
376:     <p><code>FineDiff</code> is natively better equipped to generate diff at granularity higher than line levels. An
377:         example of this is that using the above built-in sample text, for word and character-level granularity, <code>FineDiff</code>
378:         roughly executes in 25 ms and 30 ms, respectively, while <code>Text_Diff</code> roughly executes in 75 ms and
